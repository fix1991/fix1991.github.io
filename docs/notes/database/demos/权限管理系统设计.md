## RBAC：基于资源的访问控制模型

RBAC(Resource-Based Access Control，基于资源的访问控制)模型，是在基于角色的访问控制模型的基础上，引入资源表和资源-权限关系表，可以实现对某个具体资源（例如：API端点）进行细粒度的权限控制。
