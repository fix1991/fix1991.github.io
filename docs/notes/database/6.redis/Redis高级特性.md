## Redis事务

Redis事务通过以下命令实现的：

- `multi`：开始事务
- `exec`：提交事务，提交命令到队列中
- `discard`：回滚事务，清空队列中的所有命令
- `watch`：监控1个或多个键，在事务执行期间被修改过，则事务执行失败
- `unwatch`：取消对所有键的监控
